# 27 — Roadmap V1

## Purpose

Roadmap V1 defines the first production-quality version of GymCord.

## V1 Product Scope

### Member
- Account creation and onboarding
- Personalized Home Dashboard
- Today's Plan
- Workout Engine
- Nutrition goals and logging
- Progress tracking
- Messaging
- Missions, streaks, XP, and achievements
- Notifications

### Trainer
- Dashboard
- Client roster
- Client profiles
- Workout builder
- Program assignment
- Exercise library
- Progress review
- Messaging
- Calendar foundation
- Atlas assistant foundation

### Gym
- Gym dashboard
- Member directory
- Staff roles
- Membership visibility
- Scheduling foundation
- Billing visibility
- Reporting foundation
- CRM foundation

### Platform
- Multi-tenant organizations and gyms
- Authentication and authorization
- White-label theme configuration
- Billing and subscription foundation
- Analytics and telemetry foundation
- Atlas AI foundation
- Security and audit foundation

## V1 Technical Priorities

1. Stable entity model
2. Tenant-safe authentication and permissions
3. Mobile member experience
4. Trainer workflows
5. Gym admin visibility
6. Billing architecture
7. Atlas AI orchestration
8. White-label configuration
9. Testing and release discipline

## V1 Success Metrics

- Onboarding completion above 90%
- First workout started within 24 hours
- Strong workout completion rate
- Weekly active trainers
- Weekly active gym admins
- Member retention lift for pilot customers
- Successful branded customer onboarding
- Reliable billing and subscription state

## V1 Principle

V1 should be small enough to ship, strong enough to sell, and disciplined enough to scale.
